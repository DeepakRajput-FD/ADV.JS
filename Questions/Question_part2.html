7. Explain the difference between primitive and reference types in JavaScript. Provide examples for each.

Primitive types are simple data types that hold a single value, such as a number or string. Boolean: a true or false value Null: an empty value Undefined: a value that has not been set Number: a numeric value String: a sequence of characters(orderd list) Symbol : a unique identifier
Reference types are complex data types that can hold multiple values, such as an array or object. Objects: A collection of key-value pairs Arrays: An ordered list of values
